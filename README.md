# HostelEase - Hostel Management System

A web-based Hostel Management System built using ASP.NET Core MVC to manage hostels, rooms, students, billing, and deposits.

## Features

- Multi-hostel support
- Room management
- Student check-in / checkout
- Monthly billing system
- Payment tracking
- Security deposit management

## Tech Stack

Backend: ASP.NET Core MVC  
ORM: Entity Framework Core  
Database: SQL Server
